## 1. PWA(Progressive Web App)란?

### 💡 PWA의 특징

**PWA**는 웹사이트를 **앱처럼** 사용할 수 있게 해주는 기술입니다.

✅ **장점:**
- 앱스토어 등록 불필요
- 설치 즉시 사용 가능
- 자동 업데이트
- 오프라인 지원
- 푸시 알림 가능 (추후 구현)
- 빠른 로딩 속도
- 앱과 동일한 사용자 경험

❌ **일반 앱과의 차이:**
- 앱스토어에서 다운로드 불가
- 일부 네이티브 기능 제한 (카메라, 블루투스 등)
- 배터리 사용량 약간 더 높음

### 🎯 우리 앱의 PWA 기능

증산도 노서도장 앱은 다음 PWA 기능을 지원합니다:

✅ **홈 화면에 추가** - 앱 아이콘 생성
✅ **전체 화면 모드** - 브라우저 주소창 숨김
✅ **오프라인 지원** - 인터넷 없이도 일부 기능 사용
✅ **빠른 로딩** - 캐시를 통한 즉시 실행
✅ **앱 아이콘** - 증산도 로고 표시
✅ **스플래시 화면** - 앱 시작 시 로고 표시

## 2. iPhone (iOS) 설치 방법

### 📱 Safari 브라우저 사용 (필수)

⚠️ **중요:** iOS에서는 반드시 **Safari 브라우저**를 사용해야 합니다.
- Chrome, Firefox 등 다른 브라우저는 PWA 설치 불가

### 📝 설치 절차

#### **Step 1: Safari에서 사이트 접속**

1. iPhone에서 **Safari 앱** 실행
2. 주소창에 도메인 입력:
   ```
   예: https://jsdnoseo.tk
   ```
3. 사이트가 정상적으로 로딩되는지 확인

#### **Step 2: 공유 버튼 클릭**

1. 화면 하단 중앙의 **공유 버튼** (⬆️ 아이콘) 탭
2. 메뉴가 아래에서 위로 올라옴

#### **Step 3: '홈 화면에 추가' 선택**

1. 아래로 스크롤하여 **"홈 화면에 추가"** 찾기
   - 아이콘: 사각형 안에 + 표시
2. **"홈 화면에 추가"** 탭

#### **Step 4: 앱 이름 확인**

1. 앱 이름 확인:
   ```
   증산도 경주 노서도장
   ```
2. 원하면 이름 수정 가능 (예: "경주노서도장")
3. 우측 상단 **"추가"** 버튼 탭

#### **Step 5: 설치 완료**

1. 홈 화면에 앱 아이콘 자동 생성
2. 아이콘 확인:
   - 증산도 로고 표시
   - 앱 이름 표시
3. 아이콘 탭하여 앱 실행

### 📸 시각 가이드 (iOS)

```
1. Safari 실행
   📱 Safari 앱 아이콘 탭

2. 사이트 접속
   🌐 주소창에 jsdnoseo.tk 입력

3. 공유 버튼
   ⬆️ 화면 하단 중앙 버튼 탭

4. 홈 화면에 추가
   ➕ "홈 화면에 추가" 선택

5. 추가 버튼
   ✅ 우측 상단 "추가" 탭

6. 앱 실행
   📱 홈 화면 아이콘 탭
```

## 3. Android (안드로이드) 설치 방법

### 📱 Chrome 브라우저 사용 (추천)

안드로이드에서는 **Chrome 브라우저**를 추천합니다.
- Chrome, Samsung Internet, Firefox 모두 지원

### 📝 설치 절차

#### **Step 1: Chrome에서 사이트 접속**

1. Android 폰에서 **Chrome 앱** 실행
2. 주소창에 도메인 입력:
   ```
   예: https://jsdnoseo.tk
   ```
3. 사이트가 정상적으로 로딩되는지 확인

#### **Step 2: 설치 프롬프트 확인**

사이트 접속 시 자동으로 설치 안내가 나타날 수 있습니다:

**방법 A: 자동 프롬프트 (추천)**
1. 화면 하단에 설치 안내 팝업 표시:
   ```
   "증산도 경주 노서도장"을(를) 설치하시겠습니까?
   [취소] [설치]
   ```
2. **"설치"** 버튼 탭
3. 자동으로 홈 화면에 아이콘 추가

**방법 B: 수동 설치**
1. 우측 상단 **⋮ (메뉴)** 버튼 탭
2. **"홈 화면에 추가"** 또는 **"앱 설치"** 선택
3. 앱 이름 확인:
   ```
   증산도 경주 노서도장
   ```
4. **"추가"** 또는 **"설치"** 버튼 탭

#### **Step 3: 설치 완료**

1. 홈 화면 또는 앱 서랍(App Drawer)에 아이콘 생성
2. 아이콘 확인:
   - 증산도 로고 표시
   - 앱 이름 표시
3. 아이콘 탭하여 앱 실행

### 📸 시각 가이드 (Android)

```
1. Chrome 실행
   🌐 Chrome 앱 아이콘 탭

2. 사이트 접속
   🔍 주소창에 jsdnoseo.tk 입력

3. 설치 프롬프트
   💬 하단 팝업 "설치" 버튼 탭

   또는

3. 메뉴 버튼
   ⋮ 우측 상단 메뉴 → "홈 화면에 추가"

4. 앱 실행
   📱 홈 화면 아이콘 탭
```

## 4. PC (데스크톱) 설치 방법

### 💻 Windows / Mac / Linux

PC에서도 PWA를 설치할 수 있습니다!

### 📝 Chrome 브라우저 설치 절차

#### **Step 1: Chrome에서 사이트 접속**

1. PC에서 **Chrome 브라우저** 실행
2. 주소창에 도메인 입력:
   ```
   https://jsdnoseo.tk
   ```

#### **Step 2: 설치 아이콘 클릭**

1. 주소창 우측에 **설치 아이콘** (🔽 또는 ➕) 표시
2. 아이콘 클릭
3. 설치 안내 팝업 표시:
   ```
   "증산도 경주 노서도장"을(를) 설치하시겠습니까?
   [취소] [설치]
   ```
4. **"설치"** 버튼 클릭

#### **Step 3: 설치 완료**

1. Windows: 시작 메뉴에 앱 추가
2. Mac: 런치패드에 앱 추가
3. 바탕화면에 바로가기 생성 (선택 사항)
4. 독립적인 창으로 실행

### 📝 Edge 브라우저 설치 절차

1. Edge에서 사이트 접속
2. 주소창 우측 **앱 설치 아이콘** (➕) 클릭
3. **"설치"** 클릭
4. Windows 시작 메뉴에 앱 추가

## 5. 설치 후 기능

### ✨ 앱처럼 동작하는 기능

#### **1. 독립적인 앱 실행**
- 브라우저 주소창 없이 실행
- 전체 화면 모드
- 앱 전환기에 별도 앱으로 표시

#### **2. 오프라인 지원**
- 인터넷 연결 없이도 기본 화면 표시
- 이전에 본 페이지 캐시 저장
- 네트워크 연결 시 자동 동기화

#### **3. 빠른 로딩**
- 첫 실행 후 즉시 로딩
- Service Worker 캐시 활용
- 이미지 및 스타일 미리 로드

#### **4. 푸시 알림 (추후 구현)**
- 도장 소식 알림
- 새 게시글 알림
- 공지사항 알림

#### **5. 자동 업데이트**
- 앱 재시작 시 자동 업데이트
- 새 버전 자동 다운로드
- 사용자 조치 불필요

### 🔧 앱 관리

#### **앱 삭제 방법**

**iOS:**
1. 앱 아이콘 길게 누르기
2. **"앱 제거"** 또는 **"홈 화면에서 제거"** 선택
3. 확인

**Android:**
1. 앱 아이콘 길게 누르기
2. **"제거"** 또는 **"삭제"** 드래그
3. 확인

**PC (Chrome):**
1. 앱 실행
2. 우측 상단 **⋮ (메뉴)** 클릭
3. **"증산도 경주 노서도장 제거"** 선택

#### **캐시 삭제 방법**

앱이 느리거나 오류가 발생하면:

**iOS/Android:**
1. 앱 실행
2. 메뉴 → 설정 (또는 F12)
3. Application → Service Workers
4. **"Unregister"** 클릭
5. 앱 재시작

**PC:**
1. Chrome에서 사이트 접속
2. F12 (개발자 도구)
3. Application → Service Workers
4. **"Unregister"** 클릭

## 6. 문제 해결

### Q1. "홈 화면에 추가" 메뉴가 안 보여요 (iOS)

**원인:**
- Chrome/Firefox 등 Safari가 아닌 브라우저 사용
- 이미 설치된 상태

**해결 방법:**
1. **Safari 브라우저**로 다시 접속
2. 이미 설치되었는지 홈 화면 확인
3. iOS 업데이트 (iOS 11.3 이상 필요)

### Q2. 설치 프롬프트가 안 나타나요 (Android)

**원인:**
- HTTPS가 적용되지 않음
- manifest.json 오류
- Service Worker 오류

**해결 방법:**
1. 주소가 `https://`로 시작하는지 확인
2. 브라우저 캐시 삭제 (설정 → 캐시 지우기)
3. 수동 설치:
   - 메뉴(⋮) → "홈 화면에 추가"

### Q3. 앱 아이콘이 일반 웹사이트 아이콘으로 나와요

**원인:**
- 아이콘 파일 로딩 실패
- manifest.json 아이콘 경로 오류

**해결 방법:**
1. 앱 삭제 후 재설치
2. 브라우저 캐시 삭제
3. 다른 기기에서 설치 시도

### Q4. 앱 실행 시 브라우저 주소창이 보여요

**원인:**
- 앱이 아닌 브라우저 북마크로 추가됨
- manifest.json의 display 설정 오류

**해결 방법:**
1. 앱 삭제 후 올바른 방법으로 재설치
2. iOS: Safari의 "홈 화면에 추가" 사용
3. Android: "앱 설치" 또는 "홈 화면에 추가" 사용

### Q5. 오프라인에서 아무것도 안 보여요

**원인:**
- Service Worker 미등록
- 캐시가 아직 생성되지 않음

**해결 방법:**
1. 온라인 상태에서 앱을 한 번 실행하여 캐시 생성
2. 주요 페이지들을 한 번씩 방문
3. 앱 재시작

### Q6. 앱 업데이트가 안 돼요

**원인:**
- Service Worker가 이전 버전 캐시 사용

**해결 방법:**
1. 앱 완전 종료 (백그라운드에서도 종료)
2. 앱 재시작
3. 캐시 삭제 후 재설치
