## 404 에러 해결

### 문제 증상
```
오류 내용: 방명록 등록 실패
404 - Page not found: /tables/guestbook
```

### 원인
**guestbook 테이블이 데이터베이스에 생성되지 않았거나 비활성화 상태**입니다.

### 해결 방법

#### ✅ 해결 완료 (v1.7.2)
이 문제는 **이미 해결되었습니다**!
- 샘플 환영 메시지가 자동으로 추가되었습니다
- guestbook 테이블이 활성화되었습니다

#### 테스트 방법
1. 페이지 새로고침 (F5)
2. 방명록 섹션으로 이동
3. 환영 메시지가 표시되는지 확인
4. 새 방명록 작성 시도

## 제출 버튼 작동 안 함

### 문제 증상
- "방명록 남기기" 버튼 클릭 시 반응 없음
- 아무 일도 일어나지 않음

### 해결 방법

#### 1단계: 브라우저 새로고침
```
F5 키 또는 Ctrl + Shift + R (강력 새로고침)
```

#### 2단계: 필수 항목 확인
- ✅ **이름** 입력했는가?
- ✅ **메시지** 입력했는가?
- ✅ 메시지가 **500자 이하**인가?

#### 3단계: 브라우저 콘솔 확인
1. F12 키 눌러 개발자 도구 열기
2. Console 탭 선택
3. 빨간색 에러 메시지 확인
4. 에러 메시지를 관리자에게 전달

## 방명록이 표시되지 않음

### 문제 증상
- 방명록 섹션이 비어있음
- "아직 작성된 방명록이 없습니다" 메시지만 표시

### 해결 방법

#### 확인 사항
1. **네트워크 연결**
   - 인터넷 연결 확인
   - Wi-Fi 또는 데이터 연결 상태

2. **페이지 로딩**
   - 페이지가 완전히 로드될 때까지 대기
   - 로딩 표시가 사라졌는지 확인

3. **브라우저 캐시**
   ```
   Ctrl + Shift + Delete
   → 캐시된 이미지 및 파일 삭제
   ```

4. **다른 브라우저 시도**
   - Chrome, Edge, Firefox 등

## 브라우저 콘솔 확인 방법

### Windows / Linux
```
F12 키
또는
Ctrl + Shift + I
또는
우클릭 → "검사" 클릭
```

### Mac
```
Cmd + Option + I
또는
우클릭 → "요소 검사" 클릭
```

### 콘솔에서 확인할 내용

#### ✅ 정상 로그 (성공)
```
✅ 방명록 초기화 완료
방명록 로드 중... (페이지: 1)
API URL: /tables/guestbook?page=1&limit=10&sort=created_at
방명록 데이터: {data: Array(1), total: 1, ...}
방명록 로드 완료: 1개
```

#### ❌ 에러 로그 (실패)
```
❌ 방명록 테이블을 찾을 수 없습니다
방명록 로드 응답 오류: 404 Not Found
```

## 일반적인 에러 메시지

### 1. "방명록 등록 실패: 400"
**원인**: 잘못된 데이터 형식
**해결**: 
- 특수문자 제거 후 다시 시도
- 이모지 사용 자제

### 2. "방명록 등록 실패: 500"
**원인**: 서버 내부 오류
**해결**:
- 잠시 후 다시 시도
- 문제 지속 시 관리자 문의

### 3. "네트워크 오류"
**원인**: 인터넷 연결 문제
**해결**:
- 인터넷 연결 확인
- VPN 사용 시 비활성화
- 방화벽 설정 확인

### 4. "방명록 컨테이너를 찾을 수 없습니다"
**원인**: 페이지 로딩 미완료
**해결**:
- 페이지 완전히 로드될 때까지 대기
- 페이지 새로고침

## 자주 묻는 질문 (FAQ)

### Q1: 방명록을 수정하거나 삭제할 수 있나요?
**A**: 현재 버전에서는 수정/삭제 기능이 없습니다. 향후 업데이트에서 추가될 예정입니다.

### Q2: 이메일을 입력하지 않아도 되나요?
**A**: 네, 이메일과 지역은 선택사항입니다. 이름과 메시지만 필수입니다.

### Q3: 좋아요는 몇 번까지 누를 수 있나요?
**A**: 각 방명록당 1번만 좋아요를 누를 수 있습니다.

### Q4: 방명록이 삭제될 수 있나요?
**A**: 부적절한 내용이 있는 경우 관리자가 삭제할 수 있습니다.

### Q5: 방명록 작성 시 로그인이 필요한가요?
**A**: 아니요, 로그인 없이 누구나 작성할 수 있습니다.
