# 📖 방명록 사용 가이드

## 🎯 방명록이란?

방명록은 증산도 경주 노서도장을 방문한 분들이 자유롭게 소감이나 메시지를 남길 수 있는 공간입니다.

### ✨ 주요 특징

- ✅ **회원/비회원 모두 작성 가능** - 회원가입 없이도 누구나 방명록을 남길 수 있습니다
- ✅ **간편한 작성** - 이름과 메시지만 입력하면 됩니다 (이메일, 지역은 선택)
- ✅ **좋아요 기능** - 마음에 드는 방명록에 좋아요를 표시할 수 있습니다
- ✅ **정렬 기능** - 최신순 또는 좋아요순으로 정렬할 수 있습니다
- ✅ **반응형 디자인** - 모바일, 태블릿, 데스크톱 모두 지원

---

## 📝 방명록 작성하기

### 1단계: 방명록 페이지 이동

1. 웹사이트 상단 네비게이션 메뉴에서 **"방명록"** 클릭
2. 또는 URL에서 직접 `#guestbook` 섹션으로 이동

### 2단계: 정보 입력

#### 필수 입력 항목

- **이름** (필수)
  - 방명록에 표시될 이름을 입력하세요
  - 실명 또는 닉네임 모두 가능합니다
  
- **메시지** (필수)
  - 최대 500자까지 입력 가능
  - 방문 소감, 감사 인사, 질문 등 자유롭게 작성하세요
  - 실시간 글자 수 카운터가 표시됩니다

#### 선택 입력 항목

- **이메일** (선택)
  - 이메일 주소를 입력하면 답변을 받을 수 있습니다
  - 입력하지 않아도 방명록 작성 가능

- **지역** (선택)
  - 거주 지역을 입력하세요 (예: 경주시, 서울시)
  - 지역별 방문자 분포를 확인할 수 있습니다

### 3단계: 제출

1. 모든 정보를 입력한 후 **"방명록 남기기"** 버튼 클릭
2. 성공 메시지가 표시되면 완료!
3. 작성한 방명록이 목록에 자동으로 추가됩니다

---

## 👀 방명록 보기

### 방명록 목록

방명록 목록에서 다음 정보를 확인할 수 있습니다:

- **작성자 아바타** - 이름의 첫 글자
- **작성자 이름**
- **지역** (입력한 경우)
- **메시지 내용**
- **작성 날짜 및 시간**
- **좋아요 수**

### 정렬 기능

방명록을 두 가지 방식으로 정렬할 수 있습니다:

#### 📌 최신순 (기본)
- 가장 최근에 작성된 방명록부터 표시
- 새로운 방명록을 먼저 확인하고 싶을 때 사용

#### ❤️ 좋아요순
- 좋아요를 많이 받은 방명록부터 표시
- 인기 있는 방명록을 확인하고 싶을 때 사용

정렬 버튼을 클릭하면 즉시 정렬이 변경됩니다.

### 더보기 기능

- 처음에는 10개의 방명록만 표시됩니다
- 하단의 **"더보기"** 버튼을 클릭하면 10개씩 추가로 로드됩니다
- 모든 방명록을 불러오면 "더보기" 버튼이 사라집니다

---

## ❤️ 좋아요 기능

### 좋아요 표시하기

1. 마음에 드는 방명록의 하단에 있는 **하트 아이콘** 버튼 클릭
2. 하트 아이콘이 채워지고 빨간색으로 변경됩니다
3. 좋아요 수가 1 증가합니다

### 좋아요 취소하기

1. 이미 좋아요를 표시한 방명록의 **하트 아이콘** 다시 클릭
2. 하트 아이콘이 비어있는 상태로 돌아갑니다
3. 좋아요 수가 1 감소합니다

### 좋아요 제한

- 각 방명록에 1번만 좋아요를 표시할 수 있습니다
- 좋아요 정보는 브라우저에 저장됩니다 (로컬 스토리지)
- 브라우저를 닫아도 좋아요 상태가 유지됩니다

## 💡 작성 팁

### 좋은 방명록 작성 예시

#### ✅ 방문 소감
```
증산도 도장에 처음 방문했는데, 도생분들이 정말 친절하게 맞아주셔서 
감사했습니다. 태을주 수행을 배울 수 있어서 좋았고, 앞으로도 자주 
참여하고 싶습니다. 감사합니다!
```

#### ✅ 감사 인사
```
증산도 진리를 접하게 되어 감사합니다. 도전 강독 시간이 특히 유익했고, 
우주와 인간의 관계에 대해 깊이 생각해볼 수 있었습니다. 경주 노서도장 
도생 여러분 감사합니다!
```

#### ✅ 질문
```
안녕하세요! 증산도에 관심이 있어서 홈페이지를 찾아보다가 방문하게 
되었습니다. 입도 절차는 어떻게 되나요? 그리고 정기 치성 시간에 맞춰 
도장 방문이 가능할까요?
```

### 피해야 할 내용

- ❌ 비방, 욕설, 혐오 표현
- ❌ 상업적 광고 또는 스팸
- ❌ 개인정보 (전화번호, 주소 등)
- ❌ 무의미한 반복 문자
- ❌ 타인을 비하하는 내용

## 🎨 방명록 디자인 특징

### 색상 및 스타일

- **아바타**: 주홍색~금색 그라데이션
- **카드 디자인**: 흰색 배경 + 그림자 효과
- **좋아요 버튼**: 회색 → 빨간색 (활성화 시)
- **제출 버튼**: 주홍색 그라데이션

### 애니메이션 효과

- 카드 호버 시 살짝 위로 이동
- 좋아요 버튼 클릭 시 부드러운 전환
- 성공 토스트 메시지 슬라이드 업 + 펄스 효과

## 📱 모바일 사용

### 터치 최적화

- 큰 버튼 크기로 터치하기 쉬움
- 입력 필드 충분한 여백
- 스크롤 부드러움

### 반응형 레이아웃

- **모바일 (767px 이하)**
  - 입력 필드 세로 배치
  - 정렬 버튼 전체 너비
  - 제출 버튼 전체 너비

- **태블릿 (768px~1199px)**
  - 입력 필드 2열 배치
  - 적절한 간격 유지

- **데스크톱 (1200px 이상)**
  - 입력 필드 3열 배치
  - 최대 너비 제한으로 가독성 향상

## ⚙️ 기술 정보

### 데이터 저장

- 방명록은 서버 데이터베이스에 저장됩니다
- 좋아요 정보는 브라우저 로컬 스토리지에 저장됩니다

### API 엔드포인트

```javascript
// 방명록 목록 조회
GET /tables/guestbook?page=1&limit=10&sort=created_at

// 방명록 작성
POST /tables/guestbook
{
  "name": "작성자 이름",
  "message": "메시지 내용",
  "email": "optional@email.com",
  "location": "선택적 지역",
  "likes": 0,
  "published": true
}

// 좋아요 업데이트
PATCH /tables/guestbook/{id}
{
  "likes": 5
}
```

### 데이터 구조

| 필드 | 타입 | 설명 |
|------|------|------|
| id | text | 고유 ID (자동 생성) |
| name | text | 작성자 이름 (필수) |
| message | rich_text | 메시지 내용 (필수) |
| email | text | 이메일 (선택) |
| location | text | 지역 (선택) |
| user_id | text | 회원 ID (로그인 시) |
| likes | number | 좋아요 수 |
| published | bool | 게시 여부 |
| created_at | datetime | 작성일시 |
| updated_at | datetime | 수정일시 |

## 🔧 문제 해결

### 방명록이 제출되지 않아요

**원인 1**: 필수 항목 미입력
- 해결: 이름과 메시지를 모두 입력했는지 확인하세요

**원인 2**: 메시지가 500자를 초과함
- 해결: 메시지를 500자 이내로 줄여주세요

**원인 3**: 네트워크 오류
- 해결: 인터넷 연결을 확인하고 다시 시도해주세요

**원인 4**: 브라우저 콘솔 오류
- 해결: F12 키를 눌러 개발자 도구를 열고 Console 탭에서 오류 메시지 확인
- 오류 메시지를 관리자에게 전달해주세요

### 좋아요가 반영되지 않아요

**원인 1**: 브라우저 캐시 문제
- 해결: 페이지를 새로고침(F5) 해주세요

**원인 2**: 로컬 스토리지 제한
- 해결: 브라우저 설정에서 쿠키/사이트 데이터 허용 확인

### 방명록 목록이 보이지 않아요

**원인**: 아직 작성된 방명록이 없음
- 해결: 첫 번째 방명록을 작성해주세요!

## 🎉 방명록 활용 아이디어

### 1. 방문 소감 공유
도장 방문 후 느낀 점을 자유롭게 공유하세요. 다른 분들에게도 좋은 정보가 됩니다!

### 2. 감사 인사
도생분들께 감사의 마음을 전하는 공간으로 활용하세요.

### 3. 질문 및 답변
증산도에 대한 궁금한 점을 질문하고, 다른 분들의 답변을 받을 수 있습니다.

### 4. 수행 경험 나누기
태을주 수행이나 도전 강독 경험을 나누며 서로 격려할 수 있습니다.

### 5. 행사 후기
도장 행사 참여 후기를 남겨주세요. 다음 행사 참여자들에게 도움이 됩니다!
