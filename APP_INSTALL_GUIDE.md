# 📱 증산도 경주 노서도장 앱 설치 가이드

## 🎯 PWA(Progressive Web App) 앱 소개

이 웹사이트는 **PWA(Progressive Web App)** 기술로 제작되어 일반 앱처럼 사용할 수 있습니다!

### ✨ PWA의 장점

- ✅ **앱스토어 없이 설치**: Google Play나 App Store를 거치지 않고 직접 설치
- ✅ **용량 절약**: 일반 앱보다 훨씬 작은 용량
- ✅ **빠른 업데이트**: 웹사이트가 업데이트되면 자동으로 반영
- ✅ **오프라인 동작**: 인터넷 연결 없이도 기본 기능 사용 가능
- ✅ **홈 화면 아이콘**: 일반 앱처럼 홈 화면에서 바로 실행
- ✅ **전체 화면**: 브라우저 주소창 없이 앱처럼 전체 화면 사용
- ✅ **푸시 알림**: 중요한 소식을 알림으로 받을 수 있음 (추후 지원)

## 📱 Android(안드로이드) 설치 방법

### Chrome 브라우저 사용

1. **웹사이트 접속**
   - Chrome 브라우저로 웹사이트 주소 입력
   - 예: `https://your-domain.com`

2. **설치 방법 1: 자동 프롬프트**
   - 화면 하단에 "홈 화면에 추가" 배너가 나타남
   - **"추가"** 또는 **"설치"** 버튼 클릭
   - 확인 팝업에서 **"추가"** 클릭

3. **설치 방법 2: 메뉴에서 설치**
   - 오른쪽 상단 **⋮** (점 3개) 메뉴 클릭
   - **"앱 설치"** 또는 **"홈 화면에 추가"** 선택
   - **"설치"** 버튼 클릭

4. **설치 완료**
   - 홈 화면에 "경주노서도장" 아이콘이 생성됨
   - 아이콘을 터치하면 앱처럼 전체 화면으로 실행

### Samsung Internet 브라우저 사용

1. 웹사이트 접속
2. 하단 메뉴 버튼 클릭
3. **"페이지 추가"** → **"홈 화면"** 선택
4. 앱 이름 확인 후 **"추가"** 클릭

## 🍎 iOS(아이폰/아이패드) 설치 방법

### Safari 브라우저 사용 (필수)

⚠️ **중요**: iOS에서는 반드시 Safari 브라우저를 사용해야 합니다.

1. **Safari로 웹사이트 접속**
   - Safari 브라우저 실행
   - 웹사이트 주소 입력

2. **공유 버튼 클릭**
   - 하단 중앙의 **공유 버튼** (↑ 화살표) 클릭
   - 또는 상단 주소창 왼쪽의 **AA** 버튼 클릭

3. **홈 화면에 추가**
   - 메뉴에서 **"홈 화면에 추가"** 선택
   - 아이콘과 이름 확인
   - 오른쪽 상단 **"추가"** 버튼 클릭

4. **설치 완료**
   - 홈 화면에 앱 아이콘 생성
   - 일반 앱처럼 실행 가능

### iOS 설치 시 주의사항

- ❌ Chrome, Firefox 등 다른 브라우저에서는 홈 화면 추가 불가
- ✅ 반드시 Safari 브라우저 사용
- ✅ iOS 11.3 이상 버전 필요

## 💻 데스크톱(PC) 설치 방법

### Chrome 브라우저

1. 웹사이트 접속
2. 주소창 오른쪽의 **설치 아이콘** (➕ 또는 컴퓨터 모양) 클릭
3. **"설치"** 버튼 클릭
4. Windows: 시작 메뉴에 앱 추가됨
5. Mac: Applications 폴더에 앱 추가됨

### Edge 브라우저

1. 웹사이트 접속
2. 주소창 오른쪽 **⋯** (점 3개) 클릭
3. **"앱 > 이 사이트를 앱으로 설치"** 선택
4. **"설치"** 버튼 클릭

## 🔧 아이콘 이미지 준비 (관리자용)

앱을 배포하기 전에 아이콘 이미지를 준비해야 합니다.

### 방법 1: 자동 생성 (추천)

1. 브라우저에서 `/images/generate-icons.html` 파일 열기
2. **"모든 아이콘 생성 및 다운로드"** 버튼 클릭
3. 자동으로 8개 크기의 아이콘 다운로드됨
4. 다운로드한 파일들을 `/images/` 폴더에 복사

### 방법 2: 온라인 툴 사용

1. https://www.pwabuilder.com/imageGenerator 접속
2. 512x512px 원본 이미지 업로드
3. 모든 크기 자동 생성 및 다운로드
4. `/images/` 폴더에 복사

### 필요한 아이콘 크기

```
icon-72x72.png
icon-96x96.png
icon-128x128.png
icon-144x144.png
icon-152x152.png
icon-192x192.png
icon-384x384.png
icon-512x512.png
```

## 🚀 배포 방법

### 1. Publish 탭에서 배포

1. 좌측 **Publish 탭** 클릭
2. **"Publish"** 버튼 클릭
3. 자동으로 웹사이트 배포됨
4. 배포된 URL 확인 (예: `https://your-project.web.app`)

### 2. 사용자에게 URL 공유

- 📱 카카오톡, 문자 메시지로 URL 전송
- 📧 이메일로 공유
- 🔗 QR 코드 생성하여 포스터에 게시
- 📢 도장 게시판에 공지

### 3. 설치 안내

사용자들에게 다음과 같이 안내하세요:

```
📱 증산도 경주 노서도장 앱 설치 안내

1. 아래 링크를 클릭하세요
   https://your-domain.com

2. 안드로이드: Chrome에서 "홈 화면에 추가" 클릭
   아이폰: Safari에서 공유 버튼 → "홈 화면에 추가" 클릭

3. 홈 화면에 앱 아이콘이 생성됩니다!

앱스토어 설치 불필요 | 무료 | 빠른 업데이트
```

## 📊 앱 업데이트 방법

### 자동 업데이트

- ✅ 웹사이트 파일을 수정하고 재배포하면 자동 반영
- ✅ 사용자가 앱을 새로 설치할 필요 없음
- ✅ 다음 접속 시 자동으로 최신 버전 적용

### Service Worker 캐시 업데이트

새로운 기능을 추가하면:

1. `service-worker.js` 파일 열기
2. `CACHE_NAME` 버전 업데이트
   ```javascript
   const CACHE_NAME = 'jsd-gyeongju-v1.4.5'; // 버전 증가
   ```
3. 재배포

## ❓ 자주 묻는 질문 (FAQ)

### Q1. 앱스토어에서 찾을 수 없나요?
A. PWA는 웹 기반 앱으로 앱스토어 등록 없이 사용 가능합니다. 웹사이트에서 직접 설치하세요.

### Q2. 설치 용량은 얼마나 되나요?
A. 약 2~5MB 정도로 매우 가볍습니다. (일반 앱의 10분의 1 수준)

### Q3. 오프라인에서도 사용 가능한가요?
A. 네, 기본 페이지는 오프라인에서도 볼 수 있습니다. 단, 새 게시글 작성 등은 인터넷 연결이 필요합니다.

### Q4. 앱을 삭제하려면?
A. 일반 앱처럼 홈 화면에서 아이콘을 길게 눌러 삭제하면 됩니다.

### Q5. 아이폰에서 Chrome으로 설치 안 되나요?
A. iOS에서는 Safari만 PWA 설치를 지원합니다. 반드시 Safari를 사용하세요.

### Q6. 업데이트는 어떻게 하나요?
A. 자동으로 업데이트됩니다. 앱을 삭제하고 재설치할 필요 없습니다.

### Q7. 푸시 알림을 받을 수 있나요?
A. 현재는 기본 기능만 지원하며, 추후 푸시 알림 기능을 추가할 예정입니다.
