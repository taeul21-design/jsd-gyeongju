# 증산도 경주 노서도장 웹 애플리케이션

후천가을 문명시대를 준비하는 증산도 경주 노서도장의 공식 웹 애플리케이션입니다.

## 📖 프로젝트 소개

이 웹 애플리케이션은 누구나 접속할 수 있는 개방된 플랫폼으로, 증산도를 소개하고 입도를 안내하며 도장 소식을 전하는 목적으로 제작되었습니다. 회원들은 자유롭게 글과 동영상을 게시할 수 있습니다.

### 주요 목적

- 💡 증산도 교리와 가르침 소개
- 🚪 입도 절차 및 방법 안내
- 📢 도장 소식 및 공지사항 전달
- 🤝 도생들의 소통 및 정보 공유

## 🌐 API 엔드포인트

### 게시글 관련 API

```javascript
// 게시글 목록 조회 (페이지네이션)
GET /tables/posts?page=1&limit=6

// 게시글 상세 조회
GET /tables/posts/{id}

// 게시글 작성
POST /tables/posts
{
  "category": "도장소식",
  "title": "제목",
  "content": "내용",
  "author": "작성자",
  "video_url": "https://youtube.com/...",
  "thumbnail": "https://...",
  "views": 0,
  "published": true
}

// 조회수 증가 (부분 업데이트)
PATCH /tables/posts/{id}
{
  "views": 100
}
```

## 💡 사용 방법

### 방문자 (일반 사용자)
1. 웹사이트 접속
2. 네비게이션을 통해 원하는 섹션 탐색
3. 게시판에서 도장 소식 확인
4. 관심 있는 게시글 클릭하여 상세 내용 확인

### 비회원 (자유게시판)
1. 게시판 → 자유게시판 선택
2. "글쓰기" 버튼 클릭
3. 이름 입력 (필수)
4. 제목, 내용 작성
5. "게시하기" 버튼으로 등록

### 회원 (로그인 후)
1. 우측 상단 "로그인" 클릭
2. "글쓰기" 버튼 클릭
3. 카테고리 선택
   - 관리자: 모든 카테고리
   - 일반 회원: 도장소식, 행사사진, 자유게시판
4. 제목, 내용 입력
5. (선택) 이미지 URL, 동영상 URL 첨부
6. "게시하기" 버튼으로 등록

### 동영상 첨부 방법
- **YouTube**: 동영상 URL 복사하여 붙여넣기
- **Vimeo**: 동영상 URL 복사하여 붙여넣기
- **직접 업로드**: MP4, WebM, OGG 형식의 직접 링크

## 📱 PWA (Progressive Web App) 지원 (NEW!)

이 웹사이트는 **PWA(Progressive Web App)** 기술로 제작되어 앱처럼 설치하고 사용할 수 있습니다!

### ✨ PWA 기능

- ✅ **앱 설치**: 앱스토어 없이 홈 화면에 직접 설치
- ✅ **오프라인 동작**: 인터넷 연결 없이도 기본 기능 사용
- ✅ **전체 화면**: 브라우저 주소창 없이 앱처럼 실행
- ✅ **빠른 로딩**: 캐시를 활용한 빠른 페이지 로딩
- ✅ **자동 업데이트**: 웹사이트 업데이트 시 자동 반영
- ✅ **푸시 알림 준비**: 중요 소식 알림 기능 (추후 지원)

### 📱 앱 설치 방법

#### Android (안드로이드)
1. Chrome 브라우저로 웹사이트 접속
2. 화면 하단 "홈 화면에 추가" 배너 클릭
3. 또는 메뉴(⋮) → "앱 설치" 선택
4. "설치" 버튼 클릭

#### iOS (아이폰/아이패드)
1. **Safari 브라우저**로 웹사이트 접속
2. 하단 공유 버튼(↑) 클릭
3. "홈 화면에 추가" 선택
4. "추가" 버튼 클릭

#### Desktop (데스크톱)
1. Chrome 또는 Edge 브라우저로 접속
2. 주소창 오른쪽 설치 아이콘 클릭
3. "설치" 버튼 클릭

**자세한 설치 가이드**: [APP_INSTALL_GUIDE.md](APP_INSTALL_GUIDE.md) 참조

### 📱 반응형 지원

- **데스크톱**: 1200px 이상 (전체 레이아웃)
- **태블릿**: 768px ~ 1199px (조정된 레이아웃)
- **모바일**: 767px 이하 (모바일 최적화)

### 모바일 특징
- 햄버거 메뉴 네비게이션

## 🔧 디버그 및 트러블슈팅

### 로그인 문제 해결

로그인이 안 될 경우 다음 디버그 페이지를 이용하세요:

**디버그 페이지**: `debug-users.html`

#### 디버그 페이지 기능

1. **사용자 목록 확인**
   - 현재 등록된 모든 사용자 조회
   - 비밀번호 해시값 확인
   - 관리자 권한 및 활성 상태 확인

2. **로그인 테스트**
   - Admin 로그인 테스트 버튼
   - taeul21 로그인 테스트 버튼
   - 실시간 로그인 프로세스 확인

3. **비밀번호 해시 계산**
   - 입력한 비밀번호의 해시값 확인
   - DB에 저장된 해시와 비교

4. **기본 계정 생성**
   - admin 계정 자동 생성 (비밀번호: password)
   - taeul21 계정 자동 생성 (비밀번호: password)

#### 사용 방법

1. 브라우저에서 `debug-users.html` 페이지 열기
2. "사용자 목록 새로고침" 버튼으로 현재 상태 확인
3. 계정이 없으면 "기본 계정 생성" 버튼 클릭
4. "Admin 로그인 테스트" 또는 "taeul21 로그인 테스트" 버튼으로 로그인 확인
5. 문제가 있으면 콘솔 로그 확인

#### 일반적인 로그인 문제

| 문제 | 원인 | 해결 방법 |
|------|------|-----------|
| 아이디/비밀번호 오류 | 계정이 없음 | 디버그 페이지에서 기본 계정 생성 |
| 로그인 실패 | 비밀번호 불일치 | 비밀번호: `password` 확인 |
| 권한 없음 | is_admin이 false | DB에서 is_admin 값 확인 |
| 계정 비활성 | active가 false | DB에서 active 값을 true로 변경 |

#### 기본 계정 정보

| 아이디 | 비밀번호 | 이름 | 권한 |
|--------|----------|------|------|
| admin | password | 관리자 | 관리자 |
| taeul21 | password | 김정연 | 관리자 |
