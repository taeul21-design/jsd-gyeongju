## 🤔 왜 외부 호스팅을 사용하나요? {#why}

### 정적 웹사이트의 한계

이 프로젝트는 **정적 웹사이트(Static Website)**로 제작되었습니다.

**정적 웹사이트란?**
- HTML, CSS, JavaScript 파일만으로 구성
- 서버가 없음 (서버리스)
- 빠르고 안정적
- 무료 호스팅 가능

**제한사항:**
- ❌ 파일을 직접 업로드하여 저장할 수 없음
- ❌ 서버 스토리지가 없음
- ❌ 대용량 파일(음악, 동영상) 저장 불가

**해결 방법:**
- ✅ 외부 클라우드 서비스에 파일 업로드
- ✅ 직접 링크 URL 복사
- ✅ 앱에서 URL로 스트리밍 재생

## 🌟 추천 호스팅 서비스 {#services}

### 0. **네이버 MYBOX** (⚠️ 제한적 - 주의 필요)

**⚠️ 중요: 네이버 MYBOX는 음악 스트리밍을 지원하지 않습니다!**

**주의사항:**
- ⚠️ **직접 스트리밍 거의 불가능**
- ⚠️ 공유 링크가 다운로드 페이지로 이동
- ⚠️ HTML5 audio 태그에서 재생 불가
- ⚠️ CORS 정책으로 브라우저 재생 차단

**장점:**
- ✅ 네이버 계정으로 간편 로그인
- ✅ 한국 사용자에게 익숙함
- ✅ 최대 30GB 무료

**단점:**
- ❌ 음악 스트리밍용이 아님 (파일 저장용)
- ❌ 직접 재생 링크 제공 안 됨
- ❌ 항상 다운로드 페이지로 리다이렉트

**결론:**
네이버 MYBOX 대신 **Archive.org 또는 Dropbox 사용 강력 권장!**

**사용 시도 (거의 작동 안 함):**
1. https://mybox.naver.com 접속
2. MP3 파일 업로드
3. '공유' → '링크 공유' 활성화
4. 링크 복사

**⚠️ 테스트 필수:**
- 링크를 브라우저에 붙여넣기
- 다운로드 페이지가 나오면 → **사용 불가** (대부분의 경우)
- Archive.org 또는 Dropbox로 즉시 변경 필요

### 1. **SoundCloud** (⭐ 추천)

**장점:**
- ✅ 음악 전용 플랫폼
- ✅ 무료 3시간 분량
- ✅ 직접 링크 제공
- ✅ 안정적인 스트리밍
- ✅ 웹 플레이어 내장

**가격:** 무료 (3시간), Pro $4.99/월 (6시간), Pro+ $9.99/월 (무제한)

**사용법:**
1. SoundCloud 계정 생성: https://soundcloud.com
2. 'Upload' 버튼 클릭
3. 음악 파일 업로드 (MP3, WAV, FLAC 지원)
4. 제목, 아티스트 정보 입력
5. 'Save' → 'Share' → 'Embed' 탭
6. 오디오 파일 직접 링크 복사

**직접 링크 얻는 법:**
```
업로드한 곡 페이지 → Share → More → Copy Link
또는 개발자 도구에서 audio 태그의 src 찾기
```

### 2. **Google Drive** (⚠️ 제한적)

**주의사항:**
- ⚠️ HTML5 audio 태그에서 직접 스트리밍 제한적
- ⚠️ CORS 정책으로 브라우저 재생 불가할 수 있음
- ⚠️ 다운로드 링크이므로 매번 다운로드됨

**장점:**
- ✅ 무료 15GB 용량
- ✅ 간단한 사용법
- ✅ 구글 계정만 있으면 됨

**가격:** 무료 (15GB), 100GB ₩2,400/월, 200GB ₩3,600/월

**⚠️ Google Drive 재생 문제:**

Google Drive는 파일 스토리지 서비스로, 음악 스트리밍에 최적화되지 않았습니다:
1. `uc?export=download` 링크는 다운로드용
2. 브라우저 보안 정책(CORS)으로 차단될 수 있음
3. 대용량 파일 또는 트래픽 많으면 제한

**권장:**
- Google Drive보다 **SoundCloud 또는 Dropbox** 사용 권장
- 테스트용으로만 사용하고, 실제 서비스에는 전용 호스팅 사용

**사용법 (제한적):**

#### Step 1: 파일 업로드
1. Google Drive 접속: https://drive.google.com
2. 새로 만들기 → 파일 업로드
3. MP3 파일 선택하여 업로드

#### Step 2: 공유 설정
1. 업로드한 파일 우클릭 → '공유' 클릭
2. '링크가 있는 모든 사용자' 선택
3. '뷰어' 권한 유지
4. '링크 복사' 클릭

#### Step 3: 직접 링크 시도 (작동 안 할 수 있음)
```
원본 링크:
https://drive.google.com/file/d/FILE_ID/view?usp=sharing

직접 링크로 변환:
https://drive.google.com/uc?export=download&id=FILE_ID
```

**예시:**
```
원본:
https://drive.google.com/file/d/1abcDEF123xyz/view?usp=sharing

변환:
https://drive.google.com/uc?export=download&id=1abcDEF123xyz
```

**⚠️ 주의:** 이 링크가 작동하지 않으면 SoundCloud나 Dropbox를 사용하세요!

### 3. **Dropbox**

**장점:**
- ✅ 무료 2GB 용량
- ✅ 직접 링크 생성 쉬움
- ✅ 안정적

**가격:** 무료 (2GB), Plus $11.99/월 (2TB)

**사용법:**

#### Step 1: 파일 업로드
1. Dropbox 접속: https://www.dropbox.com
2. 파일 업로드

#### Step 2: 공유 링크 생성
1. 파일 우클릭 → '링크 복사'
2. 링크 끝의 `?dl=0`을 `?dl=1`로 변경

**예시:**
```
원본:
https://www.dropbox.com/s/abc123xyz/music.mp3?dl=0

변환:
https://www.dropbox.com/s/abc123xyz/music.mp3?dl=1
```

## 📝 단계별 업로드 가이드 {#guide}

### Step 1: 음악 파일 준비

**권장 사양:**
- **포맷:** MP3 (가장 호환성 좋음)
- **비트레이트:** 128-320 kbps
- **샘플레이트:** 44.1 kHz
- **파일 크기:** 3-10MB (3-5분 곡 기준)

**파일명 규칙:**
```
좋은 예: 천지성공_AI창작.mp3
나쁜 예: song123!@#.mp3
```

---

### Step 2: 호스팅 서비스에 업로드

**추천 순서:**
1. **SoundCloud** - 음악 전용, 최고의 사용자 경험
2. **Google Drive** - 무료 용량 충분, 안정적
3. **Dropbox** - 간단한 사용법

각 서비스별 상세 방법은 위의 [추천 호스팅 서비스](#services) 참조

---

### Step 3: 직접 링크 복사

**중요:** 반드시 **직접 재생 가능한 링크**여야 합니다.

**올바른 링크 형식:**
```
✅ https://drive.google.com/uc?export=download&id=FILE_ID
✅ https://www.dropbox.com/s/abc/music.mp3?dl=1
✅ https://soundcloud.com/user/track-name
```

**잘못된 링크:**
```
❌ https://drive.google.com/file/d/FILE_ID/view
❌ https://www.dropbox.com/s/abc/music.mp3?dl=0
❌ 파일:///C:/Users/음악/노래.mp3
```

---

### Step 4: 앱에서 업로드

#### 1. 로그인
- **admin** 계정 또는 **taeul21** 계정으로 로그인
- 관리자만 음악 업로드 가능

#### 2. 증산도 성곡 섹션 이동
- 네비게이션 메뉴에서 '증산도 성곡' 클릭
- 또는 스크롤하여 음악 섹션 찾기

#### 3. 음악 업로드 버튼 클릭
- 하단의 '음악 업로드' 버튼 클릭
- 모달 창이 열림

#### 4. 정보 입력

**필수 입력:**
- **곡 제목:** 예) 천지성공
- **음악 파일 URL:** 위에서 복사한 직접 링크

**선택 입력:**
- **작곡/연주자:** 예) AI 창작, 김○○
- **카테고리:** AI창작곡, 수상곡, 태을주, 찬가, 기타
- **곡 설명:** 간단한 설명
- **앨범 커버 이미지 URL:** 선택사항

#### 5. 업로드 완료
- '업로드' 버튼 클릭
- "음악이 성공적으로 업로드되었습니다! 🎵" 메시지 확인
- 음악 목록에 자동으로 추가됨

## ❓ 자주 묻는 질문 {#faq}

### Q1: 음악이 재생되지 않아요

**가능한 원인:**
1. **네이버 MYBOX 링크 문제** (가장 흔함!)
   - 문제: 네이버 MYBOX는 직접 스트리밍을 지원하지 않음
   - 증상: "지원하지 않는 형식" 오류 또는 아무 반응 없음
   - 해결: **즉시 Archive.org 또는 Dropbox로 변경** ⚠️

2. **Google Drive 링크 문제**
   - 문제: Google Drive도 스트리밍에 최적화되지 않음
   - 증상: "네트워크 오류" 또는 아무 반응 없음
   - 해결: **Archive.org 또는 Dropbox 사용**

3. **URL이 직접 링크가 아님**
   - 확인: 브라우저에서 URL을 직접 열었을 때 음악이 즉시 재생되는지
   - 해결: 위의 가이드대로 직접 링크로 변환

4. **파일 형식 문제**
   - 확인: MP3 파일인지 확인
   - 해결: WAV, FLAC 등은 MP3로 변환

5. **공유 권한 문제**
   - 확인: Dropbox에서 `?dl=1` 파라미터 확인
   - 해결: 공유 설정 다시 확인

6. **CORS 오류** (브라우저 보안)
   - 문제: 일부 서비스는 웹 브라우저에서 직접 재생 제한
   - 해결: **Archive.org 또는 SoundCloud 사용** (CORS 문제 없음)

**추천 해결 순서:**
1. **네이버 MYBOX 사용 중이면 → 즉시 Archive.org로 변경!**
2. Archive.org에 업로드 (무료, 무제한, 100% 작동 보장)
3. Dropbox 시도 (`?dl=1` 확인)
4. SoundCloud 시도
5. ❌ Google Drive와 네이버 MYBOX는 사용하지 마세요

### Q2: 무료 용량이 부족해요

**해결 방법:**
1. **여러 서비스 분산 사용**
   - Google Drive 15GB + Dropbox 2GB = 17GB

2. **음질 조정**
   - 128kbps MP3로 변환하면 용량 50% 감소
   - 온라인 변환 도구: https://online-audio-converter.com/

3. **유료 플랜 고려**
   - SoundCloud Pro: $4.99/월 (6시간)
   - Google Drive 100GB: ₩2,400/월

### Q3: YouTube 음악도 올릴 수 있나요?

**가능하지만 권장하지 않음:**
- YouTube 링크는 직접 오디오 재생 어려움
- YouTube to MP3 변환 후 Google Drive에 업로드 추천

**추천 절차:**
1. YouTube to MP3 변환 (예: https://ytmp3.cc/)
2. MP3 파일 다운로드
3. Google Drive에 업로드
4. 직접 링크 생성하여 앱에 업로드

### Q4: 여러 곡을 한 번에 업로드할 수 있나요?

**현재는 불가능:**
- 현재는 한 곡씩 업로드해야 합니다
- 각 곡마다 업로드 버튼을 눌러 정보 입력

**빠른 업로드 팁:**
1. 모든 음악을 먼저 Google Drive에 업로드
2. 직접 링크를 메모장에 미리 정리
3. 앱에서 연속으로 업로드

---

### Q5: 업로드한 음악을 수정하거나 삭제할 수 있나요?

**현재 버전에서는 불가능:**
- 음악 수정/삭제 기능은 아직 구현되지 않았습니다
- 추후 업데이트 예정

**임시 해결:**
1. 관리자 또는 개발자에게 문의
2. 데이터베이스에서 직접 수정/삭제 가능

---

### Q6: 음악 파일 형식은 무엇을 지원하나요?

**지원 형식:**
- ✅ **MP3** (권장) - 가장 호환성 좋음
- ✅ **WAV** - 고음질이지만 용량 큼
- ✅ **OGG** - 일부 브라우저에서 지원
- ✅ **M4A** - Apple 기기에서 주로 사용

**권장:**
- MP3 (128-320kbps) 사용 권장
- 모든 브라우저와 기기에서 재생 보장

---

### Q7: 저작권 문제는 없나요?

**주의사항:**
- ⚠️ 본인이 창작한 음악만 업로드하세요
- ⚠️ 타인의 음악은 저작권 허가 필요
- ⚠️ AI 생성 음악도 서비스 이용약관 확인

**안전한 음악:**
- ✅ 본인이 직접 작곡한 음악
- ✅ AI로 창작한 음악 (서비스 약관 확인)
- ✅ 증산도 관련 자체 제작 음악
- ✅ 저작권 프리 음악

---

### Q8: 네이버 MYBOX 링크를 사용하고 있는데 안 들려요!

**예상 증상:**
- "지원하지 않는 음악 파일 형식이거나 URL이 올바르지 않습니다" 오류
- 브라우저 콘솔에 "MEDIA_ERR_SRC_NOT_SUPPORTED" 오류
- 클릭해도 아무 반응 없음

**원인:**
네이버 MYBOX는 파일 저장 서비스로, **음악 스트리밍을 지원하지 않습니다.**

**즉시 해결 방법:**
1. Archive.org 계정 생성 (무료)
2. 같은 음악 파일을 Archive.org에 업로드
3. 직접 링크 복사
4. 앱에서 음악 수정 (초록색 수정 버튼)
5. URL을 Archive.org 링크로 변경
6. **즉시 재생 가능!** ✅

**Archive.org 장점:**
- 완전 무료
- 무제한 용량
- 100% 작동 보장
- CORS 문제 없음

**절대 사용하지 마세요:**
- ❌ 네이버 MYBOX (스트리밍 불가)
- ❌ Google Drive (제한적)
