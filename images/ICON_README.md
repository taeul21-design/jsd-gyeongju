# 앱 아이콘 이미지 준비 가이드

## 필요한 아이콘 크기

PWA 앱으로 만들기 위해 다음 크기의 아이콘이 필요합니다:

- 72x72px
- 96x96px
- 128x128px
- 144x144px
- 152x152px
- 192x192px
- 384x384px
- 512x512px

## 아이콘 디자인 가이드

### 추천 디자인 요소
- 증산도 심볼 또는 로고
- 한글 "증산도" 또는 "경주노서"
- 배경색: 빨간색 (#C8102E) 또는 흰색
- 텍스트 색상: 흰색 또는 금색 (#FFC72C)

## 아이콘 생성 방법

### 방법 1: 온라인 툴 사용 (가장 쉬움)
1. https://www.pwabuilder.com/imageGenerator 방문
2. 512x512px 크기의 원본 이미지 업로드
3. 모든 크기의 아이콘 자동 생성 및 다운로드
4. 다운로드한 파일들을 `/images/` 폴더에 복사

### 방법 2: Canva 사용
1. https://www.canva.com/ 에서 512x512px 캔버스 생성
2. 증산도 로고 또는 텍스트 디자인
3. PNG로 다운로드
4. 위 방법 1의 PWA Builder로 리사이징

### 방법 3: Photoshop/GIMP 사용
1. 512x512px 캔버스 생성
2. 디자인 작업
3. 각 크기별로 저장 (Image > Image Size)

## 임시 아이콘 사용

아이콘이 준비되지 않은 경우, 현재 프로젝트에 포함된 임시 아이콘을 사용할 수 있습니다.
실제 배포 전에는 반드시 정식 디자인된 아이콘으로 교체해야 합니다.
