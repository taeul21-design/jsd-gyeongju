## ⚠️ 로그인 오류 해결

### 문제: "아이디 또는 비밀번호가 올바르지 않습니다"

**해결 방법 1: 브라우저 콘솔 확인**
```
1. F12 키 누르기 (개발자 도구)
2. "Console" 탭 클릭
3. 로그인 시도
4. 콘솔에 표시되는 메시지 확인:
   - "전체 사용자 수: X"
   - "입력한 비밀번호 해시: XXXXX"
   - "검사 중: admin, 해시: XXXXX, 활성: true"
```

**해결 방법 2: 페이지 새로고침**
```
1. Ctrl + F5 (강력 새로고침)
2. 관리자 계정 자동 생성 확인
3. 다시 로그인 시도
```

**해결 방법 3: 로컬 스토리지 초기화**
```
1. F12 → Application 탭
2. Local Storage 클릭
3. 모든 항목 삭제
4. 페이지 새로고침
```

## 📋 관리자 전용 기능

### 1. 공지사항 작성

#### 방법 A: 게시판에서 직접 (추천 ⭐)
```
1. 상단 메뉴 "게시판" 클릭
2. 카테고리 필터에서 "공지사항" 클릭
3. "이 카테고리에 글쓰기" 버튼 자동 표시
4. 버튼 클릭하여 글 작성
```

#### 방법 B: 글쓰기 버튼 사용
```
1. 상단 우측 "글쓰기" 버튼 클릭
2. 카테고리: "공지사항" 선택 (관리자만 선택 가능)
3. 제목, 내용 작성
4. 이미지/동영상 URL 첨부 (선택)
5. "게시하기" 클릭
```

### 2. 글쓰기 폼 작성

## 🎨 이미지/동영상 첨부 방법

### 이미지 첨부
```
1. 이미지를 온라인에 업로드 (예: imgur.com, Google Photos)
2. 이미지 URL 복사
3. "이미지 URL" 입력란에 붙여넣기
   예: https://i.imgur.com/abc123.jpg
```

### 동영상 첨부 (YouTube)
```
1. YouTube 동영상 페이지 접속
2. 주소 복사
   예: https://www.youtube.com/watch?v=abc123
3. "동영상 URL" 입력란에 붙여넣기
4. 자동으로 임베드 처리됨
```

### 썸네일 이미지
```
- 게시글 목록에서 표시되는 미리보기 이미지
- 없으면 기본 이미지 사용
- 권장 크기: 800x600px
```

## 🔧 문제 해결

### Q1. 로그인 후에도 공지사항 글쓰기 버튼이 안 보여요
**A1.** 
1. 우측 상단에 "김정연 [관리자]" 배지가 있는지 확인
2. 없으면 로그아웃 후 다시 로그인
3. 게시판 → 공지사항 카테고리 클릭
4. "이 카테고리에 글쓰기" 버튼 확인

### Q2. 비밀번호를 변경하고 싶어요
**A2.**
현재는 데이터베이스에서 직접 변경해야 합니다.
추후 프로필 설정 기능 추가 예정입니다.

### Q3. 다른 사람의 글을 수정/삭제하고 싶어요
**A3.**
현재는 수정/삭제 기능이 없습니다.
추후 관리자 권한으로 추가 예정입니다.

### Q4. 글을 작성했는데 안 보여요
**A4.**
1. 게시판 메뉴로 이동
2. 해당 카테고리 선택
3. 최신 글 확인
4. 안 보이면 페이지 새로고침 (F5)
