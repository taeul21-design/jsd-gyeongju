## 📝 Step 1: 도메인 등록 (10분)

### 무료 도메인 (Freenom) - 추천

- [ ] https://www.freenom.com 접속
- [ ] 원하는 도메인명 입력 (예: `jsdnoseo`)
- [ ] `.tk` 또는 `.ml` 도메인 선택
- [ ] "Get it now!" 클릭
- [ ] 등록 기간 **12 Months @ FREE** 선택
- [ ] 이메일 인증 및 회원가입 완료
- [ ] "Complete Order" 클릭
- [ ] ✅ 도메인 등록 완료!

**등록한 도메인:** `__________________.tk`

📄 **상세 가이드:** `DOMAIN_REGISTRATION_GUIDE.md`

## 📝 Step 2: Netlify 회원가입 (5분)

- [ ] https://www.netlify.com 접속
- [ ] "Sign up" 클릭
- [ ] GitHub 계정으로 로그인 (또는 이메일 가입)
- [ ] ✅ 회원가입 완료!

📄 **상세 가이드:** `HOSTING_CONNECTION_GUIDE.md`

## 📝 Step 3: 파일 업로드 (10분)

### 파일 준비

- [ ] 프로젝트 폴더를 ZIP으로 압축
  ```
  jsdnoseo-app.zip
  (index.html, manifest.json, css/, js/, images/ 등 포함)
  ```

### Netlify에 업로드

- [ ] Netlify에서 "Add new site" 클릭
- [ ] "Deploy manually" 선택
- [ ] ZIP 파일 드래그앤드롭
- [ ] 배포 완료 대기 (1~2분)
- [ ] 임시 URL 확인: `https://random-name-123456.netlify.app`
- [ ] ✅ 업로드 완료!

### 사이트 이름 변경

- [ ] "Site settings" 클릭
- [ ] "Change site name" 클릭
- [ ] 원하는 이름 입력 (예: `jsdnoseo`)
- [ ] 새 URL 확인: `https://jsdnoseo.netlify.app`
- [ ] ✅ 이름 변경 완료!

---

## 📝 Step 4: 커스텀 도메인 연결 (15분)

### Netlify에 도메인 추가

- [ ] "Domain settings" 클릭
- [ ] "Add custom domain" 클릭
- [ ] 도메인 입력: `__________________.tk`
- [ ] "Verify" 클릭
- [ ] DNS 설정 안내 확인

### Freenom DNS 설정

- [ ] Freenom 로그인
- [ ] "Services" → "My Domains" 클릭
- [ ] 해당 도메인 선택 → "Manage Domain"
- [ ] "Management Tools" → "DNS Management" 클릭

#### A 레코드 추가

- [ ] Type: **A**
- [ ] Name: **@** (또는 빈칸)
- [ ] TTL: **14400**
- [ ] Target: **75.2.60.5**
- [ ] "Save Changes" 클릭

#### CNAME 레코드 추가

- [ ] Type: **CNAME**
- [ ] Name: **www**
- [ ] TTL: **14400**
- [ ] Target: **jsdnoseo.netlify.app** (본인의 Netlify 주소)
- [ ] "Save Changes" 클릭

- [ ] ✅ DNS 설정 완료!
- [ ] ⏳ DNS 반영 대기 (10분 ~ 24시간, 보통 1~2시간)

## 📝 Step 5: HTTPS 활성화 (자동, 10분~1시간)

DNS 설정이 반영되면 자동으로 HTTPS가 활성화됩니다.

### 확인 방법

- [ ] Netlify "Domain settings" 접속
- [ ] "HTTPS" 섹션 확인
- [ ] "Certificate active" 표시 확인
- [ ] "Force HTTPS" 활성화
- [ ] `https://__________________.tk` 접속 테스트
- [ ] 자물쇠 아이콘(🔒) 표시 확인
- [ ] ✅ HTTPS 활성화 완료!

⏳ **대기 시간:** 10분 ~ 1시간
