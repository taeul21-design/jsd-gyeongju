## 1. 도메인이란?

**도메인(Domain)**은 인터넷 주소입니다.
- 예: `jsdnoseo.com`, `noseodojang.kr`, `jsd-gyeongju.tk`
- 도메인이 있으면 누구나 `https://도메인주소`로 접속 가능
- **모바일 앱 설치**를 위해서는 도메인이 필수!

## 3. 무료 도메인 등록 (추천)

### 🎁 Freenom (무료 도메인)

**가장 추천하는 방법!** 완전 무료로 도메인을 등록할 수 있습니다.

#### ✨ 장점
- ✅ 완전 무료 (0원)
- ✅ 즉시 사용 가능
- ✅ 1년 무료 (매년 갱신 가능)
- ✅ DNS 설정 지원
- ✅ 신용카드 불필요

#### 📌 가능한 도메인
- `.tk` (토켈라우)
- `.ml` (말리)
- `.ga` (가봉)
- `.cf` (중앙아프리카공화국)
- `.gq` (적도 기니)

---

### 📝 Freenom 도메인 등록 절차

#### **Step 1: Freenom 접속**

1. 웹브라우저에서 접속: https://www.freenom.com
2. 언어를 영어로 설정 (한국어 없음)

#### **Step 2: 도메인 검색**

1. 메인 페이지 검색창에 원하는 도메인 입력
   - 예: `jsdnoseo` (확장자 제외)
2. **"Check Availability"** 버튼 클릭

#### **Step 3: 도메인 선택**

1. 사용 가능한 도메인 목록 확인
   ```
   jsdnoseo.tk ✅ Available (Get it now!)
   jsdnoseo.ml ✅ Available (Get it now!)
   jsdnoseo.ga ✅ Available (Get it now!)
   ```

2. 원하는 도메인 선택 (추천: `.tk` 또는 `.ml`)
3. **"Get it now!"** 클릭
4. 우측 상단 장바구니 아이콘 확인

#### **Step 4: 등록 기간 선택**

1. 장바구니에서 **"Checkout"** 클릭
2. 등록 기간 선택:
   ```
   Period: 12 Months @ FREE
   ```
3. **"Continue"** 버튼 클릭

#### **Step 5: 회원가입 / 로그인**

**신규 사용자:**
1. 이메일 주소 입력 (Gmail 추천)
2. **"Verify My Email Address"** 클릭
3. 이메일로 받은 인증 링크 클릭
4. 비밀번호 설정 및 개인정보 입력

**기존 사용자:**
1. **"Sign In"** 클릭하여 로그인

#### **Step 6: 주문 완료**

1. 이용약관 동의 체크박스 선택
2. **"Complete Order"** 버튼 클릭
3. 완료 메시지 확인:
   ```
   ✅ Order Confirmation
   Your order has been successfully completed!
   ```

#### **Step 7: DNS 설정 (나중에 진행)**

도메인 등록 후 호스팅 연결 시 DNS 설정을 진행합니다.
- 자세한 내용은 `HOSTING_CONNECTION_GUIDE.md` 참조

---

### 🔄 Freenom 도메인 갱신

무료 도메인은 **1년마다 갱신**이 필요합니다:

1. 만료 2주 전 알림 이메일 수신
2. Freenom 로그인 → **"Services" → "My Domains"**
3. 해당 도메인 선택 → **"Renew"** 클릭
4. 무료로 1년 연장 (최대 10년까지 무료)

## 4. 유료 도메인 등록

더 전문적인 도메인(.com, .kr)을 원하면 유료 도메인을 추천합니다.

  - ✅ 간편한 DNS 설정
  - ✅ 도메인 관리 UI 우수
- 결제: 신용카드, 계좌이체

#### **2) HostingKR (호스팅케이알)**

- 장점:
  - ✅ 저렴한 가격
  - ✅ 호스팅 + 도메인 통합 관리
  - ✅ 한국어 지원
- 결제: 신용카드, 계좌이체

#### **3) Cafe24 (카페24)**
- 홈페이지: https://www.cafe24.com
- 가격:
  - `.com`: 약 16,500원/년
  - `.kr`: 약 22,000원/년
- 장점:
  - ✅ 쇼핑몰 연동 가능
  - ✅ 한국어 지원
- 결제: 신용카드, 계좌이체

---

### 📝 유료 도메인 등록 절차 (Gabia 기준)

#### **Step 1: Gabia 접속 및 회원가입**

1. https://www.gabia.com 접속
2. 우측 상단 **"회원가입"** 클릭
3. 이메일 인증 후 회원가입 완료

#### **Step 2: 도메인 검색**

1. 메인 페이지 **"도메인"** 메뉴 클릭
2. 검색창에 원하는 도메인 입력
   - 예: `jsdnoseo`
3. **"검색"** 버튼 클릭

#### **Step 3: 도메인 선택 및 장바구니**

1. 사용 가능한 도메인 확인
   ```
   jsdnoseo.com ✅ 사용 가능 (15,400원/년)
   jsdnoseo.kr  ✅ 사용 가능 (22,000원/년)
   ```

2. 원하는 도메인 선택 후 **"장바구니 담기"** 클릭

#### **Step 4: 결제**

1. 장바구니에서 **"결제하기"** 클릭
2. 등록 기간 선택 (1년 추천)
3. 결제 수단 선택:
   - 신용카드
   - 계좌이체
   - 무통장입금
4. **"결제"** 버튼 클릭

#### **Step 5: 소유자 정보 입력**

1. 도메인 소유자 정보 입력:
   - 이름: 김정연
   - 단체명: 증산도 경주 노서도장
   - 주소: 경북 경주시 금성로 271, 3층(노서동)
   - 전화: 054-742-1691
2. **"등록 완료"** 클릭

#### **Step 6: 등록 완료 확인**

1. 이메일로 등록 완료 알림 수신
2. Gabia 관리 페이지에서 도메인 확인 가능

## 🆘 문제 해결

### Q1. Freenom에서 도메인이 "Taken(사용 중)"으로 나옵니다.

**해결 방법:**
- 다른 도메인명 시도 (예: `jsdnoseo2.tk`, `jsd-noseo.tk`)
- 다른 확장자 시도 (.ml, .ga, .cf, .gq)

### Q2. Freenom 회원가입 이메일이 안 옵니다.

**해결 방법:**
- 스팸 메일함 확인
- 다른 이메일 주소 사용 (Gmail 추천)
- 5분 후 다시 시도

### Q3. 도메인 등록 후 언제 사용할 수 있나요?

**답변:**
- Freenom: 즉시 사용 가능 (0~10분)
- 유료 도메인: 결제 후 10분~1시간
- DNS 설정 반영: 최대 24시간 (보통 1~2시간)

### Q4. 도메인을 여러 개 등록할 수 있나요?

**답변:**
- 가능합니다! Freenom은 계정당 최대 10개까지 무료 등록 가능
- 예: `jsdnoseo.tk`, `noseodojang.ml` 동시 등록
